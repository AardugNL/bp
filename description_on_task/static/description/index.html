Description on task

Overview

Description on task which can set a html content in description and classifie through javascript.

Description field on task. Set description content in field.

Set html content in description field on task ans set a js class that can set various event on that task.

Set html content in description field will also set the description on Kanban card which will display on dashboard.

Click on the customer name will call the javascript class and it will highlight the similar customer named task on the dashboard.

Click on the kanban tag will call the javascript class and it will classifie and highlight the similar tag named task on the dashboard.
